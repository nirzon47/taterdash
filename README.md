# taterDash - A zomato lookalike

## [API Documentation](./server/README.md)

## What is taterDash?

It is a zomato inspired food delivery app.

## What features does it have?

-  Zomato lookalike
-  User authentication
-  Cart
-  Order history (WIP)
-  REST API
-  NextJS frontend
-  Razorpay payment gateway
-  Responsive design

## What did I use for this project?

-  Typescript
-  Bun
-  NextJS
-  NodeJS
-  Express
-  MongoDB
-  Mongoose
-  Bcrypt
-  JSON Web Token
-  Razorpay
-  NextJS

-  Many Many more...
